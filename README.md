# OOP_2381
Проектная работа в рамках дисцплины "Объеткно-ориентированное программировние"
Работа представляет из себя 2-д игру с визуалицей, написанную на c++.

Игра:
Сюжет - вы играете за спартанского воина, которому необходимо стать самым сильным мужчиной в родной стране. Для этого герой хочет победить всех врагов, включая финального босса. На пути игрока встречаются различные противники, а также загадки, за решение которых игрок получает предметы (зелье силы и зелье лечения). Также на поле есть священное озеро прибыв в которое игрок восстанавливает здоровье. За каждый шаг игрок получает штрафное очко, счет очков заканчивается, когда вы убиваете финального босса.

Цель игры - убить финального босса, при этом заработав минимальное количество штрафных очков
